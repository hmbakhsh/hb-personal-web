Most design systems fail because they try to do too much too soon. Here's how to build one that actually scales.

## Start Small

Don't build a design system. Build components you need, document them, and let the system emerge.

## The Token Layer

Colors, spacing, typography. Get these right first. Everything else builds on top.

## Component Composition

The best components are primitives that compose well. Don't build a mega-button with 47 props.
